# 使用vuedraggable 完成拖拽需求

Draggable为基于Sortable.js的vue组件，用以实现拖拽功能。 项目背景：老项目新增的一个需求，要求用户通过移动端页面拖拽列表完成增加删除移动顺序等功能，因为老项目已经上线，此次需求采用 script本地引入方式。
使用技术栈：Vue3 + axios + vuedraggable + PHP

    var url = window.location.href

    const app = {
      data() {
        return {
          arr: [],
          list: '',

        axios
          .get(this.line + '.json')
          .then(response => (this.list = response.data, this.list_str = JSON.stringify(this.list)))

        axios
          .get('all_no.json')
          .then(response => (this.all_json = response.data))
      },
      methods: {
        onStart() {

        },

        onEnd() {
          this.list_str = JSON.stringify(this.list)
        },

        add(no_input, name_input) {
          if (this.no_name != '无此管理号' && this.no_name != '') {
            this.list.push({
              "no": no_input,
              "name": this.no_name
            })
            alert('添加成功')
            this.list_str = JSON.stringify(this.list)
          }
        },
        ifNo(no_input) {
          try {
            for (i = 0; this.all_json[i].no != ''; i++) {
              if ((this.all_json[i].no == no_input)) {
                this.no_name = this.all_json[i].name
                break;
              }
            }
          } catch {
            this.no_name = '无此管理号'
          }
          try {
            for (list_i = 0; this.list[list_i].no != ''; list_i++) {
              if (this.list[list_i].no == no_input) {
                alert('此线别已存在该设备，请仔细确认后添加');
                break;
              }
            }
          } catch {}
        }
      }
    }
    Vue.createApp(app).mount('#app')

拖拽部分

     <draggable :list="list"   //用于实现拖拽的数组
      group="group_1"    //用于分组，同一组的不同list可以相互拖动
      handle=".item"     //定义拖动手柄，本页面定义了一个播放按钮作为拖动手柄
      ghost-class="ghost"  //被拖动时生成影子单元占位的class
      :force-fallback="true"  //定义为true，不使用原生html5的拖放。方便定义样式
      chosen-class="chosenClass"   //目标被选中时的样式
      animation="200"      //动画时间200ms
      @start="onStart"
      @end="onEnd">
        <template #item="{ element }">
          <tr class="lasttd">
            <td style="width: 55vw;border:2px solid #fff">
              <span class="item">&nbsp;▶&nbsp;</span> No.{{ element.no }}
            </td>
            <td style="border:2px solid #fff;width: 45vw;">{{ element.name }}
            </td>
          </tr>
        </template>
      </draggable>

其中'draggable'部分是vuedraggable的标准语法里面包含了
@start="onStart" 表示开始拖动时运行函数onStart
@end="onEnd" 表示结束拖动时运行函数onEnd
此页面每次开始和结束拖动时都会更新json数组并转化为字符串。方便form表单提交

# 大功告成

第一次尝试使用Vuedraggable，意外的简单实用。再结合实际背景，充分的满足了项目需求。在这之前也尝试过原生js配合html5来写此需求，实在是太繁琐。在网上发现了这个优秀的拖拽组件，一来是稍微练手了Vuedraggable，拓宽了我的技术栈，二来是充分满足了项目需求，领导和同事用起来都很满意。一次非常开心的项目需求开发经验，历时2天。
